# typescript/

Documentation for the TypeScript engine — the headless Bonk.io physics simulation server that powers the RL environment. Covers the core simulation loop, worker pool parallelism, IPC bridge, and telemetry subsystem.

## Subdirectories

| Directory | Description |
|:----------|:------------|
| [src/](src/) | Source module docs — core engine, IPC layer, telemetry |
| [tests/](tests/) | Test runner and test suite documentation |

Documentation in this tree mirrors the TypeScript source structure under `src/`. Each `.ts` module has a corresponding `.md` file in the same relative path under `docs/typescript/`.
