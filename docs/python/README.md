# python/

Documentation for the Python client — the Gymnasium-compatible RL interface that communicates with the TypeScript physics engine over ZeroMQ.

## Subdirectories

| Directory | Description |
|:----------|:------------|
| [envs/](envs/) | Gymnasium environment wrappers |
| [utils/](utils/) | Training logging and map visualization utilities |
| [tests/](tests/) | Connectivity and load testing scripts |
| [benchmarks/](benchmarks/) | Performance benchmarking tools |
