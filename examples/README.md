# Examples

Usage examples demonstrating how to integrate and use the Bonk.io RL Environment.

## Files

| File | Description |
|------|-------------|
| `parallel-envs.ts` | Demonstrates running multiple environments concurrently for parallel data collection |

## Running Examples

All examples are written in TypeScript and can be run directly with `npx tsx`:

```bash
npx tsx examples/parallel-envs.ts
```
