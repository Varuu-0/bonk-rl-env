# Reference (`reference/`)

Reference code from the original Bonk.io game, kept for understanding original physics behavior, verifying constants, and debugging edge cases.

## Contents

| Directory | Description |
|-----------|-------------|
| `bonk1-box2d/` | Original ActionScript 3 Box2D physics engine from Bonk 1 (Flash) |

## Note

This code is **NOT compiled or used** by the RL environment. The active project uses the `box2d` npm package (JavaScript port of Box2D). This reference material exists solely for understanding the original game mechanics.
